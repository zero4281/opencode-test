# opencode-test

## Installation
### On macOS/Linux
`python3 -m venv .venv`

### On Windows
`python -m venv .venv`

### On macOS/Linux
`source .venv/bin/activate`

### On Windows
`.venv\Scripts\activate`

### Install the Hugginface Hub API
`pip install huggingface_hub`

## Run the Code
`source .venv/bin/activate`  
`python hf_search.py -r model -a Jackrong`
